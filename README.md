# Circuit Analysis Application

A web-based DC circuit analysis tool for electrical engineering education and professional use.

## Features

- Visual circuit editor with drag-and-drop components (resistors, voltage sources, current sources)
- Automatic nodal analysis (cut-set method) and loop analysis (tie-set method)
- Step-by-step mathematical solutions with formatted matrices and equations
- Multi-circuit management in a single session

## Tech Stack

- React 19 + TypeScript
- Vite build tool
- Zustand state management
- React Flow visual editor
- KaTeX math rendering
- Bun runtime

## Development

```bash
bun install          # Install dependencies
bun run dev          # Start development server
bun run build        # Build for production
bun run lint         # Check code quality
```
